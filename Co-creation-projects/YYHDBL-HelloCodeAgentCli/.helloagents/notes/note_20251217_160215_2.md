# 项目结构分析

## 项目结构概览

**项目类型**: Python AI Agent 框架/工具包
**主要特征**:
1. 模块化设计 - 清晰的目录结构
2. 虚拟环境已配置 (.venv)
3. 版本控制 (.gitignore)
4. 环境变量配置 (.env)
5. 依赖管理 (requirements-*.txt)

**目录结构**:
- `agents/` - Agent 实现
- `code_agent/` - 代码相关 Agent
- `context/` - 上下文管理
- `core/` - 核心框架
- `memory/` - 记忆管理
- `tools/` - 工具库
- `utils/` - 工具函数
- `docs/` - 文档

**依赖文件**:
- requirements-mvp.txt - 最小可行产品依赖
- requirements-optional.txt - 可选依赖

**配置**:
- .env - 环境变量
- .helloagents - 项目特定配置
- version.py - 版本信息
